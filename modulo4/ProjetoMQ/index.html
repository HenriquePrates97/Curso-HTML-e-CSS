O que é media query?

É uma técnologia de adaptação do conteúdo de um site para resoluções menores ou para outros tipos de utilização como impressão, por exemplo. São definidas através da estilização via CSS.

Media Types

É possível fazer links de CSS específico para cada situação que usaremos a estilização, como por exemplo, temos um link para quando o arquivo for impresso, e outro para o arquivo que será exibido normalmente no site. Em cada estilo, podemos modificar da forma que acharmos melhor para cada utilização, retirando conteúdos que não são relevantes na hora da impressão e mantendo esses mesmos conteúdo para a exibição no site, assim tendo uma estilização mais enxuta na hora de imprimir.

Para fazer essa distinção de utilização, juntamente ao link da CSS, utilizamos o parametro media="" e dentro dos parenteses colocamos o o valor desejado, neste caso, print é para impressão, e screen é para tela de exibição no site, all é para uma configuração geral, serve tanto para impressão, quanto para exibição.

Media Features

Assim como as media types, as media features servem para alterar a estilização em determinado tipo de situação. Uma das formas é selecionar uma media feature para atuar de acordo com a orientação da tela.

As media features são declaradas juntamente com as media types, dentro das aspas. Após a declaração da media type, escrevemos and e declaramos a media feature entre parenteses, ficando assim: media="screen and (orientation: landscape).
