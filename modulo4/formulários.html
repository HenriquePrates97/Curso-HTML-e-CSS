Primeiro de tudo, todo formulário deve estar dentro da tag form, assim como toda tabela deve estar dentro da tag table, para que a semantica do HTML possa ser seguida a risca. Também é muito importante que haja uma ligação entre o campo destinado a inserção de informações e a etiqueta que o determina, ou seja, quando colocamos uma área para o usuário digitar seu nome, a instrução "Nome" deve ter uma ligação com a caixa de texto utilizada para digitar o nome, e isso é feito com o uso da tag label. O nome do campo a ser preenchido deve estar dentro da tag label, que terá o parametro for, dentro do parametro for, colocamos o id do input que escolhemos para essa seção. Um detalhe: para o for, apenas o id do input é reconhecido, o name do input não é reconhecido no for.
