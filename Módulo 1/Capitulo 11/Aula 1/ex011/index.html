Imagem dinâmica

Tente abrir este site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador

a tag picture é utilizada para abrir imagens que se ajustarão de acordo com a largura do navegador no usuário, onde utiliza-se a tag source para chamar as imagens menores, que devem ser chamadas quando o tamanho da largura da tela atingir determinado limite, no caso, esse limite é determinado em media max-width colocando ao lado dos dois pontos o valor maximo em pixels que a largura deve ter para mostrar determinada imagem. Dentro da tag img fica a imagem bruta, no tamanho maior desejado para mostrar ao usuario. Dentro do parametro srcset é colocado o nome da imagem a ser mostrada, o atalho ctrl + espaço não funciona para selecionar a fonte de imagem nesse caso, é necessário escrever normalmente.

Reproduzindo Áudio

Vamos apender a reproduzir áudios em HTML5

para adicionar uma música supimpa ao seu site, existem duas maneiras. A primeira seria utilizando a tag audio /audio onde dentro de src seria colocada a fonte do audio, alguns parametros como autoplay (para a musica começar a tocar automaticamente, mas pode não funcionar em todos os navegadores), controls para aparecer o layout de controle do som, com botão de play, pause, volume... e o paremtro loop para o som tocar em loop assim que acabar. a segunda maneira também utiliza a tag audio /audio porém não usamos o src que vem dentro da tag, e sim criamos uma tag source dentro da tag audio /audio, como no exemplo acima. No parametro type colocamos o tipo de midia, como já vimos em aulas passadas, nesse caso, o tipo da midia audio é audio/mpeg.

Reproduzindo Vídeos

para adicionar vídeos usamos a tag video para fazer isso, juntamente com o comando controls para aparecer os menus de play e pause assim como fazemos na inserção de audio.

o correto é sempre colocarmos no site o maior numero de extensões possiveis para que o usuario não tenha probelma de compatibilidade se o navegador não suportar determinado tipo de video.

para isso, abrimos a tag video e dentro dela colocamos uma source com todos os formatos possiveis, dando preferencia para o formato com o menos tamanho primeiro, e ao final um paragrafo para exibir uma mensagem caso o navegador não seja compativel com nenhum tipo de formato. O comando controls, witdh, e outros ficam junto à tag video.
