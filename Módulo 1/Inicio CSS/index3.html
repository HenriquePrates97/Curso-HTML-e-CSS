Capitulo 1

O metodo externo consiste em criar um arquivo .css, de preferencia com o nome style, para chamar este arquivo no codigo atraves da tag link:css e assim fazer com que os estilos fiquem em um arquivo separado e não poluam o codigo e tenham uma facilidade maior de serem alterados para o codigo inteiro. É possivel utilizar os outros métodos de CSS no mesmo código, sendo o inline o com maior relevancia, seguido pelo local e por ultimo o externo. Se eu quiser colocar um h1 com determinado estilo em todas as páginas usando o CSS externo e em uma página especifica colocar esse mesmo h1 com outro estilo, é possivel usando o CSS inline, ou se todos os h1 dessa mesma página forem diferentes, é possivel usar o metodo interno para modifica-lo.

Para o metodo externo é necessário colocar a tag link:css em todas as páginas, esta tag deve ser escrita na parte head do código.

Também é possivel utilizar mais de 1 arquivo css no codígo, apenas é necessário criar mais um link:css e adicionar o caminho do outro arquivo.
